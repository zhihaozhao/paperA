# Excel/LibreOffice Calc 绘图操作指南
# IEEE IoTJ Figure 3 & 4 手工绘制步骤

## Figure 3: Cross-Domain Generalization Performance

### 数据准备:
1. 打开新的Excel工作表
2. 输入以下数据结构:
   ```
   A      B        C           D        E
   Model  LOSO_F1  LOSO_Error  LORO_F1  LORO_Error
   Enhanced  0.830  0.001      0.830    0.001
   CNN       0.842  0.025      0.796    0.097
   BiLSTM    0.803  0.022      0.789    0.044
   Conformer 0.403  0.386      0.841    0.040
   ```

### 绘图步骤:
1. 选择数据区域 A1:E5
2. 插入 -> 图表 -> 柱状图 -> 聚集柱形图
3. 右键图表 -> 选择数据 -> 添加误差线

### 格式设置:
- 图表标题: "Cross-Domain Generalization Performance" (12pt, 加粗)
- X轴标题: "Model Architecture" (10pt)  
- Y轴标题: "Macro F1 Score" (10pt)
- Y轴范围: 0 到 1.0
- 字体: 全部改为 Times New Roman

### 颜色方案 (色盲友好):
- Enhanced系列: RGB(46,134,171) - #2E86AB 深蓝色
- CNN系列: RGB(232,72,85) - #E84855 橙红色
- BiLSTM系列: RGB(60,179,113) - #3CB371 中绿色  
- Conformer系列: RGB(220,20,60) - #DC143C 深红色

### Enhanced模型突出:
- 右键Enhanced柱状 -> 格式化数据系列 -> 边框 -> 黑色, 1.5pt宽度

## Figure 4: Sim2Real Label Efficiency Curve

### 数据准备:
1. 输入数据:
   ```
   A            B         C
   Label_Ratio  F1_Score  Std_Error
   1.0          0.455     0.050
   5.0          0.780     0.016
   10.0         0.730     0.104  
   20.0         0.821     0.003
   100.0        0.833     0.000
   ```

### 绘图步骤:
1. 选择A1:C6数据
2. 插入 -> 散点图 -> 带平滑线和数据标记的散点图
3. 添加误差线: 选择数据系列 -> 右键 -> 添加误差线 -> 自定义 -> 使用C列

### 参考线添加:
1. 右键图表 -> 添加趋势线/参考线:
   - 目标线: y = 0.80 (红色虚线, 1.5pt)
   - 理想线: y = 0.90 (橙色点线, 1pt)  
   - Zero-shot: y = 0.151 (灰色实线, 1pt)

### 关键标注 (20%, 0.821点):
1. 插入 -> 文本框
2. 位置: 图表右上方 (35%, 87%位置)
3. 文本: "Key Achievement:\n82.1% F1 @ 20% Labels"
4. 格式: Times New Roman, 10pt, 加粗
5. 背景色: RGB(255,250,205) - #FFFACD 浅黄色
6. 边框: RGB(255,107,107) - #FF6B6B 红色, 1pt
7. 添加箭头: 插入 -> 形状 -> 箭头 -> 指向(20%, 0.821)

### 格式设置:
- 图表标题: "Sim2Real Label Efficiency Breakthrough" (12pt, 加粗)
- X轴: 对数刻度, 标题"Label Ratio (%)" (10pt)
- Y轴: 线性刻度0.1-0.95, 标题"Macro F1 Score" (10pt)
- 主曲线: RGB(46,134,171) - #2E86AB, 2.5pt宽度
- 数据标记: 圆形, 8pt大小

### 效率区域背景:
1. 插入 -> 形状 -> 矩形
2. 位置: X轴0-25%区域覆盖
3. 填充: RGB(153,255,153) - 浅绿色, 20%透明度
4. 无边框

## 输出规范:
- 分辨率: 300 DPI
- 格式: PDF (矢量) 或 高分辨率PNG
- 尺寸: Figure 3: 17.1cm × 10cm, Figure 4: 17.1cm × 12cm
- 字体检查: 确保全部为 Times New Roman

## 常见问题解决:
1. 误差线不显示: 检查数据列是否正确选择
2. 颜色不准确: 使用RGB值而非近似色
3. 字体不一致: 选中所有文本元素批量修改
4. 图表尺寸问题: 使用页面设置精确控制
5. 导出模糊: 确保300DPI设置生效
